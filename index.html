SRM · RTM V1 相對趨勢計算方法
透明公式
  1. 相對價格 rt = ln(AdjCloseasset,t / AdjClosebenchmark,t)
  2. h日趨勢 Mt,h = rt − rt−h
  3. 平滑趨勢 Tt = EMA(Mt,h, s), s = max(5, round(h/4))
  4. 趨勢加速度 At = Tt − Tt−s
  5. 標準化座標 X = 100 + clip(2 × T / RMS252(T), −4, 4)
    Y = 100 + clip(2 × A / RMS252(A), −4, 4)
01領先 · X≥100 / Y≥100 02改善 · X<100 / Y≥100 03落後 · X<100 / Y<100 04弱化 · X≥100 / Y<100

X 代表相對趨勢，Y 代表相對趨勢加速度；100為象限中心。此為自家透明方法，並非 JdK RRG 或 XQ 指標複製。
